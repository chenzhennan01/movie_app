##  项目功能
- 选票
- 登录
- 查看排片
- 查看影片详情
- 查看排片情况

##  下载运行
1.下载到本地
</br>
2.在终端输入命令行 ``npm install`` 下载项目依赖
</br>
3.输入``npm run dev``启动程序
